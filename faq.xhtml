Frequently asked (stupid) questions
  1. Q: Who/what/how/why/when [is] Omploader?
    A: Omploader was originally created to become the ultimate shock site hosting service by Brenden Matthews and David Shakaryan, but has since evolved into much more. Omploader is written almost entirely in Ruby, which was chosen for no particular reason (actually David was a Ruby fanboy, and Brenden didn't care). All of the omploader code is available from the svn repository (see footer).
  2. Q: Are there any restrictions or limits on content?
    A: There is a 64MiB file size restriction, and there are no restrictions on content. However, we retain the right to delete whatever the crap we feel like if we find it offensive, irritating, lame, wasteful, or whatever.
